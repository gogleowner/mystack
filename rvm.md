# RVM (Ruby Version Manager)

- Ruby Version Manager
- https://rvm.io
- `pyenv` 처럼 루비 버전을 한 머신에 여러 버전을 설치하고 관리할 수 있게끔 해주는 툴이다.

# Usage
- Install RVM

```
$ \curl -sSL https://get.rvm.io | bash -s stable
```

- Upgrade RVM version

```
$ rvm get stable
```

- Install Ruby

```
$ rvm install [ruby-version]
```
