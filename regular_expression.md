# Regular Expression

## 전방탐색, 후방탐색

- 전방탐색 : `?=`

    - 예제문장 : `https://image.server.url/some-path-variable/20150914.jpg` 
    - `.jpg` 로 끝나는 파일에서 숫자를 가져오기 : `\d+(?=.jpg)`
    - 프로토콜 가져오기 : `\w+(?=:)`

- 후방탐색 : `?<=`

    - 예제문장 : `ABC01: $23.45`
    - $ 뒤의 숫자 가져오기 : `(?<=\$)[0-9.]+`

## 그룹
- 그룹에서 제외(non-capturing group) : `(?:{패턴})`
    - 예제 : `([0-9])(?:.*)(블라블랍)`
        - 그룹1 -> `[0-9]` 패턴
        - 그룹2 -> `블라블랍` 패턴

- 그룹에 그룹명 지정
